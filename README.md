# Multi-Container App

A simple full-stack application with **React frontend**, **Node.js backend**, and **MongoDB database**.

## Features
- Node.js API with Express
- MongoDB database
- React frontend
- Dockerized services
- CI/CD pipeline with GitHub Actions

## Requirements
- Docker
- Docker Compose
- Node.js (for local development)
- npm

## How to Run Locally
1. Clone the repository:
   ```bash
   git clone <your-repo-url>
   cd multi-container-app
   docker-compose up --build
